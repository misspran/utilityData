# Description
Simple dashboard of utility data per month with charts and savings of energy bill.

# Setup
- npm install
- npm run start begin

# Deployed Link
Site can be found at:

http://utility-data.surge.sh/

# Technologies
-React
-React Router Dom
-VictoryJS 
-Semantic UI React
